# OpenDrop

> Community-owned, open-data map of every clothing donation location in the United States — drop bins, thrift stores, mutual aid closets, church drives, and seasonal collection points. Civic infrastructure, not a product.

**Status:** 🚧 Under active construction. This project is being built in phases per [`AGENTS.md`](AGENTS.md):

1. **Phase 1 — Research & Verification** → [`research/FINDINGS.md`](research/FINDINGS.md)
2. **Phase 2 — Architecture & Planning** → [`planning/`](planning/)
3. **Phase 3 — Construction** → `backend/`, `frontend/`, `pipeline/`
4. **Phase 4 — Final Validation** → `FINAL_VALIDATION.md`

A full setup-to-running-map guide will land here when Phase 3 completes. Until then, the canonical source of intent is [`AGENTS.md`](AGENTS.md).
